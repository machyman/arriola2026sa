\noindent\textbf{How to use this manual.}
Solutions are organized by chapter and exercise number, matching the
labeling in the main text.  For each exercise, the Bloom's Taxonomy
level is given in brackets.  L2 solutions are concise; L5--L6 solutions
provide model answers that can be adapted for grading rubrics.
Where MATLAB verification is natural, it is included.

\medskip
\noindent For exercises marked \textit{Instructor note}, the note
provides grading guidance, common student errors, or pedagogical context.
